# Chart Design System

Welcome to the Chart Design System Storybook. This is a comprehensive collection of data visualization components across 6 major JavaScript charting libraries.

## ðŸ“š Available Libraries

### Recharts
React-specific charting library built on D3.js with composable components and declarative API.
- Line Charts
- Bar Charts
- Pie & Donut Charts
- Area Charts
- Radial & Radar Charts
- Composed Charts

### D3.js
Low-level, powerful data visualization library for creating custom, interactive visualizations.
- Sankey Diagrams
- Force-Directed Graphs
- Radial Trees
- Hierarchy Charts
- Custom Visualizations

### Plotly.js
Scientific computing library with 3D capabilities, geographic maps, and statistical charts.
- Geographic Heatmaps
- 3D Surface Plots
- Bubble Charts
- Waterfall Charts
- Sunburst Charts
- Parallel Coordinates

### Three.js
3D graphics library for creating animated, interactive 3D visualizations in the browser.
- 3D Pipeline Visualizations
- 3D Clustering
- Animated Meshes
- Interactive 3D Dashboards

### Chart.js
Simple, flexible JavaScript charting library with responsive, animated charts using Canvas.
- Line Charts
- Bar Charts
- Doughnut & Pie Charts
- Radar Charts
- Bubble & Scatter Plots
- Mixed Charts

### ApexCharts
Modern charting library with extensive customization options and built-in interactivity.
- Line & Area Charts
- Column & Bar Charts
- Heatmaps
- Radial Charts & Gauges
- Sparklines
- Candlestick & OHLC Charts
