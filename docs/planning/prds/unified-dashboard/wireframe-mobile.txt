KEY FEATURES (MOBILE):
- Hamburger menu (☰) for navigation drawer
- Collapsible tab selector dropdown
- Condensed metrics bar
- Vertical pipeline flow with arrows
- Touch-optimized cards and buttons
- Infinite scroll for prospect list
- Simplified visualizations
- Full-width search and filters

MOBILE INTERACTIONS:
- Tap hamburger menu to open navigation
- Tap tab dropdown to switch views
- Tap pipeline stages to filter
- Swipe cards for additional actions
- Pull-to-refresh for updates
- Tap prospect cards for details
- Tap campaign cards to pause/resume

RESPONSIVE BREAKPOINTS:
- < 768px: Mobile single column
- 768px - 1024px: Tablet two columns
- > 1024px: Desktop full layout
