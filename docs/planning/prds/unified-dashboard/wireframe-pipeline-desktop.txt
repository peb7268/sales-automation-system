KEY FEATURES:
- Tab-based navigation with active tab highlighting
- Real-time metrics bar showing key performance indicators
- Four metric cards with icons and trend indicators
- Interactive pipeline stages that can be clicked to filter
- Search and filter controls
- Split view: Funnel visualization on left, prospect list on right
- Research monitor showing API health status
- Action buttons for Export, Research, and Add Prospect

INTERACTIONS:
- Click pipeline stages to filter prospects
- Search bar for finding specific prospects
- Industry dropdown filter
- Clear filters button to reset view
- Real-time toggle (🔄) for live updates
- Notification bell with unread count
- Theme toggle (sun/moon icon)
